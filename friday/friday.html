Bootstrap is configured with some fixed colors like warning, success, infor, primary, secondary among others

there are different sizes of screen fitting. Instead of using media querry one can use this

    types of sizes and property representation using container
  1. .container > very small. this means it fits on the small mobile screen
  2. .container-sm > small. This fit in the small container
  3. .container-md > medium. This fit in the medium mobile screen and tablets
  4. .container-lg > large. This fit in large screen size
  5. .container-xl > this fit in extra large screen size
  6. .container xxl > This fit in the extra extra large screen size
